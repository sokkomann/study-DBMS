Redis(Remote Dictionary Server)
	빠른 속도의 메모리 기반 키-값 저장소로 주로 캐시, 세션 저장, 토큰 관리 등에 활용된다.
	이를 인메모리(in-memory)(=캐싱) 데이터베이스라고 한다.

Redis 사용목적
	매우 빠른 읽기/쓰기 속도(메모리 기반)
	데이터 만료기능(TTL) 지원
	분산 시스템에 적합 => 트래픽이 몰릴때 L4S로 분산된 서버에 맞게 지정후 session통합
	여러 데이터 타입 지원(문자열, 리스트, 해시, 집합 등)

Redis 활용
	1. 분산 시스템 환경에서의 세션 공유
	2. 로그아웃 혹은 강제 로그아웃 시 토큰 무효화	
	        =>	로그인 할때 토큰생성해서 레디스에저장
		로그아웃 할때도 토큰생성해서 레디스에 저장
		(더이상 사용하지 않는 토큰을 블랙리스트에 저장함)
	3. 캐싱: DB조회한 결과 또는 외부 API 호출 결과 등 비용이 큰 작업 저장 후 빠른 응답 처리
